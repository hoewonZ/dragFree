# dragFree 使用说明（功能总览）

本文面向使用者，介绍 dragFree 的主要功能、用法与配置项说明。若某处适合配图，已用 **`[截图占位：……]`** 标出，可自行截图补充。

## 一、应用是什么、解决什么问题

**dragFree** 是一款在 Windows 桌面上工作的 **文件拖拽路由工具**：把文件拖到屏幕上的 **热区**，再进入 **目标文件夹选择面板**，把文件 **复制或移动** 到你指定的常用目录（或当前浏览到的子目录）。  
同时支持 **纯文本/链接类内容** 的拖入与展示编辑、将 **当前标签页文本导出为 Word（.docx）**，以及 **快速打开目录（快开）** 等辅助能力。

`[截图占位：配置窗口主界面（含文件夹列表与齿轮按钮）]`

### 3.2 常用文件夹列表

**能做什么**

- 维护一组你经常投放文件的 **目标文件夹**。
- 每个条目通常包含 **名称** 与 **路径**（路径过长时会以省略方式展示，避免撑爆界面）。
- 支持 **拖拽排序**：通过拖动手柄调整常用文件夹顺序（顺序会影响面板里展示/使用时的优先级体验）。
- 列表区域在条目较多时支持 **边缘自动滚动**，方便拖到很上/很下的位置。

**如何添加**

- 点击 **“+ / 添加文件夹”**（以界面实际按钮为准）选择目录加入列表。

**如何删除/变更**

- 按界面提供的删除/编辑入口管理条目（具体以你当前版本 UI 为准）。

**提示**

- 某些操作后会短暂出现 **“已添加 / 已删除 / 已更改”** 一类提示，用于提醒本次会话内的变更感知。

`[截图占位：常用文件夹列表含拖动手柄与省略路径效果]`

### 3.3 自动保存与“为什么有时像没生效”

**自动保存**

- 多数配置项修改后会 **自动保存**（无需手动点“保存文件”这类步骤）。
- 底部状态文字会提示保存或失败原因（以实际提示为准）。

**（1）哪些配置可能不会“立刻”在你期望的界面里体现出来**

- **热区外观、展示文本、背景图等**：改完后一般会尽快反映到屏幕热区，但若你刚做过 **跨屏移动、系统缩放/显示器变更** 等，偶尔会出现热区画面与预期不一致；此时可先 **折叠/展开热区** 或 **切换一次显示器**（多屏时）再观察。
- **开机自启动**：属于系统登录启动能力，通常 **不会在这一次会话里立刻改变“是否已经自启”的全部观感**；更合理的验证方式是：**下次开机/重新登录后** 看是否随系统启动。
- **与系统目录、权限相关的操作**（例如选择文件夹失败、目标盘不可写）：这类问题不属于“配置没保存”，而是 **当前环境不允许**，需要换路径或以管理员身份等（以系统提示为准）。

**（2）哪些情况会导致“其实没保存成功”，下次打开像没改过**

- 本应用多数配置会 **自动保存**，但保存并不是“你每点一下就立刻落盘一次”的机械节奏；若在保存尚未完成时发生 **异常退出**（例如崩溃、强制结束进程、断电关机等），可能出现 **最后一次改动丢失**。
- 因此：如果你刚改完一批设置就立刻结束电脑/强制关闭应用，**下次启动发现回退**，优先怀疑是 **上次未正常完成保存**；正常关闭（退出菜单）一般更可靠。

## 五、显示设置（热区外观）

### 5.1 热区颜色、标题栏颜色、热区透明度

- **热区颜色**：热区主体区域的配色。
- **标题栏颜色**：热区顶部标题栏区域配色。
- **热区透明度**：调节热区整体透明程度（数值为 0–1 的滑杆，界面会显示当前值）。

通常同时支持：

- **调色板选色**（色块按钮）
- **十六进制输入**（如 `#RRGGBB`）

**关于系统调色板取色**

- **系统调色板取色**会受到运行环境限制：一般 **无法从屏幕任意位置吸取颜色**（例如无法像某些专业取色工具那样指着别的窗口取色）；同时调色板窗口 **不一定能按你的习惯完全隐藏或不影响操作**。
- 若你需要精确颜色，建议优先使用 **十六进制输入（#RRGGBB）** 或先用外部工具取色后再粘贴。

## 六、弹窗设置（与面板/快开相关）

### 6.1 面包屑分隔符

- 用于目录面板/快开界面里 **路径面包屑** 的展示样式（例如 `/` 或 `>`）。

### 6.2 悬停查询延时（毫秒）

- 控制鼠标悬停在目录项上后，多久开始触发 **子目录查询/展开** 类行为（用于减少误触与频繁查询）。

- 两次查询之间的 **冷却间隔**，用于避免短时间内重复查询带来的卡顿或干扰。

### 6.4 快开悬停延时（毫秒）

- **快开模式**下，与“悬停后再进入下一步动作”相关的延时（与上一条不同：面向快开交互）。

### 6.5 投放成功后打开目标文件夹

- 勾选后：当文件成功投放到目标位置，可 **自动打开目标文件夹**（便于立刻查看结果）。

### 6.6 默认处理方式：复制 / 移动

- 作为拖拽投放时的 **默认策略**：

### 6.8 平铺尺寸（面板与快开共用）

- 平铺模式下提供 **大/中/小** 等档位（影响每行显示数量，例如“3列/5列/7列”这类体验）。

## 八、屏幕热区（Overlay）：触发区 + 记事中转

热区的 **位置是自定义的**，核心作用是提供一个 **稳定、好找的触发区域**：你把文件拖到这里，就可以进入后续的目录选择与投放流程。

由于应用窗口在桌面上的工作方式，热区 **必然会遮挡其下方的屏幕内容**。为了不把这块区域只当作“空白遮挡”，产品在这里叠加了 **轻量的文本记录能力**（展示文本、标签页、拖入文本/链接等），相当于在热区里承担了一部分 **临时记事/中转** 的用途，让遮挡区域更有价值。

`[截图占位：热区遮挡桌面内容 + 文本区/标签条的关系（表达“遮挡但可记录”）]`

### 8.1 标题栏常见按钮（从左到右/自适应折叠）

> 注：窗口变窄时，部分按钮可能收纳到 **“更多”菜单**，但能力仍存在。

常见能力包括：

- **置顶/取消置顶（📌）**  
  - 控制热区是否以“置顶”方式保持在最前（便于始终可拖拽命中）。

- **折叠 / 展开（▴ / ▾）**  
  - 折叠后热区收起，仅保留标题栏/窄条（具体视觉以实际为准）。  
  - 若出现跨屏移动后界面异常，界面提示可用 **折叠按钮** 作为刷新手段。

- **切换显示器**  
  - 多显示器时用于在显示器之间切换热区所在屏幕；仅单显示器时通常会提示不可用。

- **文本大小（A- / A+）**  
  - 调整展示文本字号档位（多档，界面按钮到达极限会禁用）。

- **导出为 Word（分享图标）**  
  - 将 **当前标签页** 的展示内容以 **纯文本** 导出为 **`.docx`**，通过 **系统「另存为」** 对话框选择保存位置与文件名。  
  - 详细说明与多屏提示见下文 **「8.1.1 导出为 Word」**。

- **交互模式切换：拖拽模式 / 快开模式**  
  - **拖拽模式**：面向“把文件投放到目标文件夹”的主流程。  
  - **快开模式**：面向“快速打开目录浏览”的流程（与右键触发等有关，见后文）。

- **锁定按钮（🔒）**  
  - 用于进入/退出 **编辑布局模式**（移动热区、调整大小等）。锁定后避免误拖误改。

### 8.1.1 导出为 Word（.docx）

**能做什么**

- 导出的是 **当前激活标签页** 中的文本（**纯文本**，保留换行；不携带富文本格式）。
- 文件格式固定为 **`.docx`**（Word 文档）。

**如何保存**

- 点击标题栏 **分享图标** 后，会打开 **系统自带的「另存为」** 对话框，由你在其中选择目录、输入文件名并确认保存。
- **默认保存位置**会落在应用配置目录下的 **`favorite-links`** 文件夹附近（便于你直接进入该目录）。该文件夹在 **Windows** 上会 **随「常用文件夹」配置同步**：其中为指向各常用目录的 **快捷方式（.lnk）**，修改常用文件夹并保存配置后，其中的快捷方式会随之增删更新；你可从系统对话框进入这些快捷方式再导航到真实路径。
- 若目标位置已存在同名文件，系统或应用会按流程提示 **是否覆盖**；若目录无写入权限等，会 **提示错误原因**，不影响热区其它功能。

**多显示器时对话框可能「闪一下」**

- 少数环境下，系统「另存为」窗口可能在 **某一瞬间** 先出现在 **主屏或其它屏幕**，再 **移动到热区所在显示器**。这是 **Windows 与系统对话框** 的显示策略所致，**不影响** 最终保存路径与文件内容；若未看清可再点一次分享重新打开。

### 8.5 拖入文本与链接（拖拽模式下）

当拖入的不是文件而是文本/链接类内容时，应用会按规则解析并 **追加到展示文本**（是否默认换行受配置项影响）。

**为什么这样设计**

很多时候我们会在浏览器里浏览资料，想把 **片段素材、可编辑文案** 先收集起来，再去别的工具里处理；如果每次都 **窗口间复制粘贴**，成本高、也容易打断心流。热区提供一个 **临时中转站**：把文字拖进来先暂存、整理，再继续下一步。

同理，**链接**也常用于“先收集、后访问”：把感兴趣的网址先丢进热区里 **记录与回溯**，减少在浏览器与纯文本工具之间来回复制粘贴。

### 8.6 调整热区大小与移动（编辑布局模式）

- 进入可编辑布局后，热区边缘会出现 **拖拽调整尺寸** 的命中区域。
- **文本编辑过程中** 通常会禁用尺寸拖拽，避免冲突（属于保护交互）。

## 九、目录面板（投放选择目标路径）

当你把文件拖入热区并进入流程后，会出现 **半透明圆角面板**，用于选择目标目录。

### 9.1 面包屑导航（悬停为主）

目录面板顶部的路径 **面包屑以悬停为主**：把鼠标移到某一级路径上，会按产品设计展开/导航（减少点击次数）。这也是 dragFree 在 **拖拽弹窗** 与 **快开弹窗** 里强调的体验：**尽量用悬停弹窗减少操作步数**，让用户更少点、更少切焦点。

`[截图占位：悬停面包屑展开/导航的关键瞬间]`

### 9.2 目录列表与悬停查询

- 列表中展示子目录等内容。
- **悬停查询延时**、**查询冷却** 会影响“悬停后多久查询/多久能再查”。

面板内提供 **新建文件夹并保存** 入口：会弹出独立窗口让你输入文件夹名称并创建。

## 十一、典型使用路径（建议照做一遍）

### 路径 A：把文件投放到常用文件夹

1. 在配置里先添加至少一个 **常用文件夹**。  
2. 从资源管理器拖动文件到 **热区**。  
3. 在面板中选择目标目录（或进入子目录）。  
4. 确认投放结果提示是否符合预期。  
5. 若开启“投放成功后打开目标文件夹”，检查是否自动打开。

`[截图占位：从拖到热区到面板出现的关键帧拼图]`

### 路径 B：使用快开模式快速浏览目录

1. 在热区标题栏切换到 **快开模式**（或 Ctrl+Space）。  
2. 在热区内容区 **右键** 触发快开。  
3. 在快开窗口内浏览与打开。

## 十三、你可能关心的边界与建议（非技术）

- **跨屏拖动热区**：可能带来异常显示；遇到问题优先尝试 **折叠/展开** 或关闭该选项。  
- **复制 vs 移动**：移动会改变源文件位置；不确定时优先用复制。  
- **文本拖入与链接格式**：若链接识别异常，可尝试开启“拖入文字先换行”的选项。  
- **调试日志**：仅排查问题时短期开启。
